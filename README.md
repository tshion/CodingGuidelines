# CodingGuidelines
> コーディング規約をまとめたリポジトリ

## 使い方
下記に対してカスタム指示を整備しているので、依頼してみてください。

* Gemini
* GitHub Copilot

### コーディング規約のファイル追加
1. チャットを開く
1. チャットに`/add-guidline` を入力し、実行する
1. 生成されたファイルにコーディング規約を追記する
